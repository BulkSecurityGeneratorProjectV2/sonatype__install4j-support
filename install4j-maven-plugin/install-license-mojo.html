install4j:install-license

Full name:

org.sonatype.install4j:install4j-maven-plugin:1.1.2:install-license

Description:

Update the install4j license key (via install4jc --license). Execution will skip if licenseKey parameter is not configured, or the install4j.home location is invalid.

Attributes:

  • Requires a Maven project to be executed.
  • Since version: 1.0.

Optional Parameters

Name Type Since Description
failIfMissing boolean 1.0 Fail if the installation is missing.
Default value is: false.
User property is: install4j.failIfMissing.
installDir File 1.0 The location of the install4j installation.
User property is: install4j.home.
licenseKey String 1.0 install4j license key.
User property is: install4j.licenseKey.
skip boolean 1.0 Skip execution.
Default value is: false.
User property is: install4j.skip.
